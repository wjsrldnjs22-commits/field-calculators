📦 CBM / 부피중량 계산기

박스 겉치수(cm) + 수량만 넣으면 총 CBM(m³)과 부피중량(kg)을 계산해줘.
※ 운송사마다 분모(5000/6000)·반올림 규칙이 다를 수 있어. 실제 청구 기준은 계약/요율표를 우선으로 봐줘.

🧮 입력

🧠 계산 로직
• 총 CBM(m³) = (가×세×높이×수량) ÷ 1,000,000
• 부피중량(kg) = (가×세×높이×수량) ÷ 분모(5000/6000 등)

✅ 빠른 체크(검산)
50×40×30cm, 1개라면
• CBM = 0.060 m³
• 분모 6000이면 부피중량 = 10.0 kg
• 분모 5000이면 부피중량 = 12.0 kg
🧯 현장 실수 방지(초안)
• “상품”이 아니라 겉박스 기준으로 재기
• cm ↔ m 단위 착각 금지(특히 0 하나!)
• 수량 곱하기 빼먹지 말기
• 여러 박스면 박스별로 따로 계산 후 합치기
